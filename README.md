## ‚ú¶ What is it ?

**AutoFollowUser** is a [BetterDiscord](https://betterdiscord.app/) plugin that lets you **automatically follow a user** whenever they switch voice channels ‚ÄĒ on the same server or any other.

No more manually joining every time. Right-click ‚Üí Follow. Done.
